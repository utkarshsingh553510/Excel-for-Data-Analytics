2) MATCH FUNCTION  

- Match Function kisi Value ki Postion Batata hai 

Syntax-> =MATCH(Lookup_value,lookup_array,0)

- Lookup_value -> Jo value Search Krni hai

- lookup_array -> Jis Range me Dhudhana hai 

- o-> Exact Match 


Q1-> Employee column me Vivek Kumar search kro 

-- employee B column me hai 

=MATCH("Vivek kumar",B1:B51,0)
