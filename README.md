## 📖 About This Repository

This repository contains my structured learning notes on Microsoft Excel. It covers essential Excel concepts required for Data Analytics, including Lookup Functions, Pivot Tables, Statistical Functions, Power Query, and more.

The goal of this repository is to serve as a quick reference guide while learning Excel and preparing for interviews.
